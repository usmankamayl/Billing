# 💰 Lectrum — Billing App

> Приложение для оплаты коммунальных услуг и счётчиков, разработанное на **чистом JavaScript (Vanilla JS)** без использования сторонних библиотек и фреймворков.

## ✨ Особенности проекта

- ⚡ **Pure JavaScript** — никаких зависимостей, только нативный JS
- 💧 **Счётчики воды** — ввод текущих/предыдущих показаний, автоматический расчёт суммы к оплате
- 🧾 **Налоги** — раздел для оплаты налоговых платежей
- 🌐 **Интернет** — оплата интернет-услуг
- 🔒 **Охрана** — оплата услуг охраны
- 💱 **Обмен валют** — конвертация и работа с валютой
- 💾 **Сохранение платежей** — платежи сохраняются для повторного использования
- ✅ **Транзакции** — отображение статуса транзакций (успешно / ошибка)
- 🗑️ **Очистка данных** — сброс введённых показаний

---

## ⚙️ Функциональность счётчиков воды

```
Выбор счётчика → Ввод предыдущих показаний → Ввод текущих показаний
       ↓
Автоматический расчёт суммы к оплате
       ↓
Сохранение / Очистка
```

**Поддерживаемые счётчики:**
- ДС 949321 — $621
- КА 522010 — $902
- ЛК 309123 — $173
- ДЛ 309123
- ВС 123309
- ПЕ 331209
